# XLSX Analysis Deliverables

Use `.xlsx` for data analysis, dashboards, action trackers, creator matrices, content matrices, material testing tables, and campaign schedules.

## Recommended Worksheets
- `Executive Summary`: top conclusions, key metrics, risks.
- `Raw Data`: supplied source data or pasted structured data.
- `Metric Analysis`: calculated metrics and comparisons.
- `Content Matrix`: content/creator/material-level performance.
- `Action Tracker`: priority, action, owner suggestion, deadline, affected metric.
- `Missing Data`: fields required for better judgment.

## Common Columns
### Metric Analysis
- 日期/周期
- 平台
- 消耗
- GMV
- ROI
- 曝光/播放
- 点击
- CTR
- 转化率/CVR
- 成交订单
- 客单价
- 备注/口径

### Content Matrix
- 素材/达人/内容标题
- 内容类型
- 核心钩子
- 主卖点
- 曝光/播放
- 点击/互动
- 加购/成交
- ROI/CVR
- 判断
- 下一步动作

### Action Tracker
- 优先级
- 动作
- 负责人建议
- 影响指标
- 所需数据
- 时间节点
- 状态

## Rules
- Do not calculate missing metrics without source values.
- Use `待补数据` for unavailable fields.
- Keep a summary sheet even when raw data is incomplete.
