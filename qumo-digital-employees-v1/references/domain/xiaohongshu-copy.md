# Xiaohongshu Copy

Use for Xiaohongshu notes, beauty seeding copy, titles, hooks, soft-selling copy, and before/after copy rewrites.

## Style Principles
- Start from a real user scene or problem.
- Keep tone conversational, specific, and experience-based.
- Avoid hard-sell phrasing.
- Use careful claims and leave space for personal suitability.
- Add titles, openings, body version, tags, and image suggestions when useful.

## Copy Structure
1. Scene hook
2. User pain point or moment
3. Product role
4. Texture/use experience/proof
5. Suitability
6. Light purchase cue
7. Comment or collection prompt

## Better Phrases
- `如果你也有...`
- `更适合...`
- `我会把它放在...这个步骤`
- `不是猛药路线，更像日常...`
- `活动期可以关注一下`

## Avoid
- `闭眼入`, `必买`, `救命`, `一夜变好`, `永久`, `根治`, `医美级`.
- Negative comparison that attacks competitors.
- Over-specific efficacy without proof.

## Deliverable Preference
- Short copy: in-chat preview is enough.
- Multiple versions or before/after explanation: `.docx`.
- Large copy matrix: `.xlsx`.
